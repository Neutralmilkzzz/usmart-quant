# Finnhub 字段可实现因子

## 用途

本文件用于回答当前 assignment 里的核心问题：

- 现有 Finnhub 可取字段，能拼出哪些可实现因子
- 哪些因子可以直接上线
- 哪些因子受限于当前权限，暂时无法实现

## 结论先行

按当前 API key 已实测可取字段，已经可以做出一批可直接落地的美股截面因子，主要集中在：

- 估值因子
- 盈利能力因子
- 成长因子
- 杠杆与偿债因子
- 规模因子
- 流动性因子
- 中短期动量因子
- 52 周位置类因子

但以下两类能力当前仍然不足：

- 依赖自建历史价格序列的时序因子
- 依赖逐笔、盘口、分钟级 feed 的微观结构因子

根因是当前 key 无法访问 `stock/candle`，也还没有验证更强的实时 feed 接口权限。

### 8. 波动与风险因子

可用字段：

- `beta`
- `3MonthADReturnStd`

可实现因子：

- Beta 因子
- 3 月收益波动因子

说明：

- 这组风险因子能直接用于组合风险暴露控制

### 9. 52 周位置与区间空间因子

可用字段：

- `52WeekHigh`
- `52WeekLow`
- `c`

可实现因子：

- 距 52 周高点比例：`c / 52WeekHigh`
- 距 52 周低点比例：`c / 52WeekLow`
- 52 周区间位置：`(c - 52WeekLow) / (52WeekHigh - 52WeekLow)`
- 52 周振幅因子：`52WeekHigh / 52WeekLow - 1`

说明：

- 这里的 `c` 来自 `quote` 当前价
- 这类因子对趋势延续和反转研究都很常见

## 当前不能可靠实现的因子

### 1. 历史 K 线衍生因子

当前缺口：

- `stock/candle` 无权限

因此暂时不能稳定实现：

- MA5 / MA20 / MA60
- 均线乖离率
- RSI
- MACD
- 布林带
- ATR
- 历史收益波动率
- 过去 N 日最大回撤

### 2. 高频 / 微观结构因子

当前缺口：

- 未验证逐笔成交 feed
- 未验证盘口深度 feed
- 未验证分钟级序列权限

因此暂时不能稳定实现：

- Order imbalance
- Bid-ask spread
- Tick imbalance
- VWAP 偏离
- 分钟级量价相关因子

### 3. 真正的时序 alpha 迭代

当前缺口：

- 截面字段多，历史时序字段不足

因此当前更适合：

- 做截面排序因子
- 做横截面多因子选股
- 做日频或低频再平衡框架

## 对纳指 / 美股股票池的直接建议

若以当前权限尽快开工，优先建议构建以下三组基础因子：

- 价值组：`peTTM`、`pb`、`psTTM`、`evEbitdaTTM`
- 质量组：`roeTTM`、`grossMarginTTM`、`operatingMarginTTM`、`netProfitMarginTTM`
- 动量组：`13WeekPriceReturnDaily`、`26WeekPriceReturnDaily`、`52WeekPriceReturnDaily`

再加两组辅助约束：

- 规模 / 流动性约束：`marketCapitalization`、`10DayAverageTradingVolume`、`3MonthAverageTradingVolume`
- 风险约束：`beta`、`3MonthADReturnStd`、`totalDebt/totalEquityQuarterly`

这样已经足够搭出一个可运行的美股多因子原型。
